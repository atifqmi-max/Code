🎯 Overview

DevStudio is a complete authentication system that provides:

· Glassmorphism Design - Modern frosted glass effect with animated background
· User Authentication - Secure login and registration system
· Session Management - Persistent sessions using localStorage
· Device Selection - Choose between Mobile and PC development environments

✨ Features

🔐 Authentication

· Login System - Secure email/password authentication
· Registration System - Create new user accounts
· Password Visibility Toggle - Show/hide password functionality
· Session Persistence - Stay logged in until logout
· Form Validation - Real-time input validation
· Demo Account - Pre-configured demo user for testing

🎨 UI/UX

· Glassmorphism Effect - Blurred glass card design
· Animated Background - Floating gradient orbs with smooth animations
· Responsive Design - Works on all screen sizes (mobile, tablet, desktop)
· Smooth Transitions - Elegant hover effects and animations
· Status Messages - Clear success/error notifications

📱 Device Integration

· Mobile Environment - Redirects to mobile.html
· PC Environment - Redirects to pc.html
· Session Validation - Ensures only authenticated users can access
